Flexbox y Grid

¿Qué es HTML Semántico?

HTML semántico consiste en usar etiquetas que describen claramente el significado del contenido, mejorando la accesibilidad, el SEO y la comprensión del código.

<header>

Define el encabezado de una página o sección.

<section>

Agrupa contenido relacionado dentro de la página.

<footer>

Contiene información final como derechos de autor.

CSS Flexbox

Flexbox es un modelo de diseño unidimensional que permite organizar elementos en filas o columnas de manera flexible.

display: flex

Activa Flexbox en un contenedor.

justify-content

Controla la alineación horizontal.

align-items

Controla la alineación vertical.

CSS Grid

CSS Grid es un sistema de diseño bidimensional que permite crear estructuras complejas con filas y columnas.

grid-template-columns

Define el número y tamaño de las columnas.

gap

Establece el espacio entre filas y columnas.

auto-fit

Hace el diseño adaptable a cualquier pantalla.
